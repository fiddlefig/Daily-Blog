About my python homework 11/5/23
So my python assignment this week was a little hard. We had to create two functions, one that would take a list of temperature in C and translate them to F, then display them side by side like
'the temperature is #C or #F'. Then the second function needed to put a weather statement next to each temperature reading depending on the temp. I first was having issues trying to get the list to iterate through the math that would translate the temperature, but i eventually got it.
